== Description ==

<p>WPShop is a free but powerful online store plugin, with the integration of various popular payment systems and PayPal. It works on any themes and it is easy to configure and use.</p>

<h3>Key features of the online store WP-Shop plugin: </h3>
<ul>
<li>Products are based on regular posts of WordPress, with addition of the price tag and the button "buy".</li>
<li>The plugin has convenient tools for creating "showcases" of goods and price lists, with various settings and adaptive layout </li>
<li>Off-line methods  of payment  and delivery: Pickup from the store / warehouse / office; Cash on  courier delivery, Cash on delivery at the post office.</li>
<li>Customizable delivery options: you can construct any delivery options instead of standard cases.</li>
<li>Online payment  methods : Webmoney, Yandex Money(directly), all kinds of e-money + VISA and MASTERCARD, through Sberbank (directly) and many popular payment systems</li>
<li>Integration with payment systems: Yandex payment system (Yandex Kassa), Robokassa (RoboKassa),  WalletOne (W1, WalletOne), PayPal, Sofort, ICredit, SimplePay, Chronopay, ArtPay, Interkassa</li>
<li>Integration with crypto-payment systems: Bitcoin BTC, Bitcoin Cash BCH, Blackcoin BLK, Bytecoin BCN, Dogecoin DOGE, Dash DASH, Emercoin EMC, Litecoin LTC, Monero XMR, Peercoin PPC, Primecoin XPM, Reddcoin RDD, Ripple XRP, Zcash ZEC
<li>Integration with delivery systems: Yandex delivery system (Strig, MaxiPost, DPD, Axiomus), Boxberry</li>
<li>Personal  customer account: orders review, purchase history, customer profile</li>
<li>Sale of "digital goods": download files after payment, with protection from unauthorized downloads or send a link to download the file to the buyer's email </li>
<li>Generating  Yandex.XML file for translation goods in Yandex. Market and other similar services </li>
<li>Integrated Google Analytics and Yandex Metrika statistics for online retailers: considers sale, the average check, and other specific statistics for online shopping. </li>
<li>Integrated with <a href="https://udsgame.com/?ref=9037143" target="_blank">UDS (United Digital Solutions, ex. UDS GAME) complex loyalty system</a></li>
<li>many other options ... </li>
</ul>

== Installation ==

1. Upload plugin "WP Shop" to the `/wp-content/plugins/` directory
2. Upload plugin "cForms2"  to the `/wp-content/plugins/` directory
3. Activate the plugin cForms through the 'Plugins' menu in WordPress
4. Activate the plugin WP Shop through the 'Plugins' menu in WordPress
5. See [quick userguide](https://wp-shop.ru/helpdesk/bystroe-podklyuchenie-internet-magazina-na-wordpress-3-prostyx-shaga/) how to set up your shop

See also:<ul>
<li>1. Full guide about WP-Shop-site creation - https://wp-shop.ru/full_video_guide/</li>
<li>2. Full guide about WP-Shop-site usage - https://wp-shop.ru/helpdesk/</li>
<li>3. DEMO-shops catalog - http://demo.wp-shop.ru/</li>
<li>4. Нuge (1800+) catalog of templates for online stores and sites on Wordpress + WP Shop - https://wpshop.store/</li>

</ul>
